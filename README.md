## About Me

I build practical, polished, and self-hosted web products. My work spans user
interfaces, plugin development, responsive behavior, deployment, automation,
testing, and technical documentation.

My featured open-source work is **CBS Mail**, a modern webmail experience built
on Roundcube for the Cybrense ecosystem.

## Featured Project

### CBS Mail

I created and developed **CBS Mail** as a branded, mobile-ready Roundcube
webmail experience. The project started in my personal GitHub space and is now
published and maintained under the
[Cybrense IT Services organization](https://github.com/Cybrense-IT-Services).

CBS Mail keeps Roundcube as the mail engine and adds a complete interface and
plugin layer with responsive desktop and mobile views, PWA support, Docker
deployment, synchronized user labels, and privacy-aware remote content.

## What I Built

| Area | My work on CBS Mail |
| --- | --- |
| **Interface** | Custom Cybrense/CBS visual system across login, mail, compose, contacts, and settings. |
| **Responsive UI** | Desktop, tablet, and mobile layouts designed for an app-like webmail experience. |
| **Roundcube Plugin** | Plugin hooks, asset loading, trusted senders, trusted domains, and user preference integration. |
| **Labels** | Per-user synchronized email labels with assignment, filtering, counts, and migration support. |
| **PWA** | Manifest, installable app experience, icons, safe offline fallback, and service worker behavior. |
| **Deployment** | Docker image overlay, Compose setup, environment templates, and production guidance. |
| **Quality** | Browser tests, security checks, CSS checks, CI workflows, and project documentation. |
