### CBS Mail

I designed and developed **CBS Mail**, an open-source Roundcube webmail experience created for Cybrense.

My work on the project includes:

- A Roundcube plugin and branded UI layer
- Mobile and PWA support
- Docker-based setup and deployment
- Per-user email labels and browser-side enhancements
- Project documentation, testing, and maintenance workflows

The project is now hosted under the **Cybrense IT Services** organization.
