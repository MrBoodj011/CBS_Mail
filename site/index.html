Le bon moteur, une meilleure expérience

Roundcube, repensé pour le travail moderne

CBS Mail conserve Roundcube comme client IMAP et ajoute une couche d'interface complète: navigation claire, vues mobiles, étiquettes synchronisées et experience proche d'une application native.

Fonctionnalités

Tout ce qu'il faut pour un webmail quotidien

Une interface cohérente sur la connexion, la messagerie, la rédaction, les contacts et les paramètres.

01

Responsive par conception

Une boite de reception lisible sur ordinateur, tablette et téléphone, avec navigation adaptee au toucher.

02

Étiquettes synchronisées

Ajout ou retrait en un clic, plusieurs étiquettes par message et synchronisation dans les préférences utilisateur.

03

PWA installable

Manifest, icones, service worker et comportement web app pour un accès rapide depuis le bureau ou l'écran d'accueil.

04

Confidentialité visible

Les ressources distantes restent bloquées par défaut, avec une autorisation claire par message ou expéditeur.

05

Notifications navigateur

Alertes de nouveaux messages via les fonctions officielles de Roundcube, configurables pour chaque compte.

06

Déploiement Docker

Image reproductible, vérification de santé, configuration par environnement et scripts de sauvegarde.

Mobile et PWA

Le même webmail, vraiment utilisable sur téléphone

Les listes, outils et messages changent de disposition selon l'espace disponible. Les actions importantes restent accessibles sans zoom ni défilement horizontal.

  • Cartes de messages tactiles
  • Navigation mobile avec tiroir
  • Dates, étiquettes et états toujours visibles
  • Installation sur l'écran d'accueil

Architecture

Une couche claire, pas un nouveau serveur mail

CBS Mail est un reskin et une couche d'amélioration pour Roundcube. Votre infrastructure IMAP et SMTP reste indépendante.

  1. Navigateur / PWAExpérience utilisateur
  2. CBS Mail UISkin, plugin et étiquettes
  3. RoundcubeClient webmail IMAP
  4. IMAP / SMTPServeur de messagerie

Démarrage rapide

Lancez CBS Mail avec Docker

Clonez le projet, configurez votre serveur IMAP/SMTP dans .env, puis démarrez la stack.

Lire le guide d'installation
PowerShell / Bash 
git clone https://github.com/MrBoodj011/CBS_Mail.git
cd CBS_Mail
cp .env.example .env
docker compose up -d --build
